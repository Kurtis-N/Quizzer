What are the two implementations of IPsec and how do they differ? AH (Authentication Header) which provides integrity, and ESP (Encapsulated Security Packet) which provides integrity and optional encrpytion.
What are the two modes of using an implementation IPsec? Transport Mode - which appends IPsec headers onto the IP headers but before the rest of the packet.  Tunnel Mode - Creates a new IP header with IPsec header ontop of the IP packet.
